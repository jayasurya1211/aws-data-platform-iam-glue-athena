## Overview
This project demonstrates a production-style AWS data platform built using
IAM, Amazon S3, AWS Glue, and Amazon Athena.

The focus is on:
- Least-privilege IAM design
- Secure S3 data lake architecture
- ETL using AWS Glue (Glue 4.0)
- Analytics using Athena on Parquet data
- Real-world debugging and validation
